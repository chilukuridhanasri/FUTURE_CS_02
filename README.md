# Phishing Email Detection & Awareness Report

## 📌 Objective
To analyze phishing emails and identify common attack patterns.

## 🛠️ Tools Used
- VirusTotal
- MXToolbox
- Google Header Analyzer

## 📧 Email Samples
Phishing email samples are available in the samples folder.

## 🔍 Analysis Approach
- Checked email content for phishing indicators
- Verified links using VirusTotal
- Analyzed headers using MXToolbox and Google Header Analyzer

## ⚠️ Key Findings
- Emails used urgency and fear tactics
- Suspicious links were detected
- Fake domains were used

## 🛡️ Conclusion
Phishing attacks can be identified through proper analysis and user awareness.
